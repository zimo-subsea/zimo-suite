# ZiMO Suite

ZiMO Suite is a desktop application built with PySide6 (Qt for Python).

## Prerequisites

- Python 3.10+ (recommended)
- A virtual environment tool such as `venv`

## Setup

From the repository root:

```bash
python -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
pip install PySide6
```

## Run the application

From the repository root:

```bash
python -m zimo.app.main
```

This launches the ZiMO Suite desktop window.
